# E-Commerce App
-----

Spring Security, Spring Boot, java 8, H2, Angular 11

# Build
-----

1. Clone this repo

2. Open the directory product-manager

3. Open Powershell

4. Build the image

   docker build -t e-commerce-frontend .
   
5. Run the image 

   docker build -t e-commerce-frontend .

6. Navigate to http://localhost:4200/login
